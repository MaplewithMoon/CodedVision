# CodedVision
CodedVision for Classification (2018)
This project realized a joint system for image coding and classification.
The classification task can be conducted immediately at the recieving side, without fully decoding the bit-stream.
The detailed introduction and implementation can be seen in our paper: 
<CodedVision: Towards Joint Image Understanding and Compression via End-to-End Learning>
https://link.springer.com/chapter/10.1007/978-3-030-00776-8_1
